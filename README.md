This project has been created as part of the 42 curriculum by ahavrank and gvasylie.

The cub3D project is inspired by Wolfenstein 3D game and is supposed to teach us mainly about raycasting.
The goal was to make a dynamic view inside a maze.

for installation:
git clone https://github.com/42paris/minilibx-linux.git
make

main resources:
https://harm-smits.github.io/42docs/projects/cub3d
https://devabdilah.medium.com/3d-ray-casting-game-with-cub3d-7a116376056a
https://hackmd.io/@nszl/H1LXByIE2

for change of view use left and right keys of keyboard
movement possible through W, A, S, D

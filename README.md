### Description
The cub3D project is inspired by Wolfenstein 3D game and is supposed to teach us mainly about raycasting.
The goal was to make a dynamic view inside a maze.

---

### Instructions
Compilation & execution

- make, make re

- ./cub3d [path_to_map.cub]
